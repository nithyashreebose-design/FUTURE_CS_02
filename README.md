# Task 2 - API Security Risk Analysis Report

* ReqBin
* Browser Developer Tools
* OWASP API Security Top 10
* Canva (for report design)

##Deliverables
* API Security Risk Analysis Report (PDF uploaded in /reports folder)

* Risk assessment and remediation recommendations

* Understanding authentication and authorization controls
* Applying the OWASP API Security Top 10 framework
* Analyzing API responses and security configurations
